# Интернет-магазин «В один клик» - персонализированные предложения
Требуется разработать решение для персонализированных предложений клиентам на основе анализа данных, с использованием моделей машинного обучения.

## Используемая модель и результаты
- **Модель:** SVC (ядро rbf)
- **Масштабирование:** StandardScaler()
- **Метрика на кросс-валидации:** 0.908
- **Метрика ROC-AUC на тестовой выборке:** 0.91
